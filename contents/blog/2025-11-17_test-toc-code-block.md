## はじめに

この記事は、コードブロック内の見出しが目次として認識されないことをテストするために作成されました。

### テストの目的

以下のようなコードブロック内の見出しが目次に含まれないことを確認します。

## 正常な見出し1

これは通常の見出しで、目次に含まれるべきです。

### 正常な見出し1-1

これも通常の見出しで、目次に含まれるべきです。

## コードブロックのテスト

以下のコードブロック内の見出しは目次に含まれてはいけません：

```md
## これはコードブロック内の見出し
このテキストはコードブロック内にあります。

### これもコードブロック内の見出し
目次に含まれてはいけません。
```

## 正常な見出し2

コードブロック後の見出しは正常に認識されるべきです。

### 正常な見出し2-1

YAMLのコードブロックでもテストします：

```yaml
# コメント
## タイトル
内容
### サブタイトル
詳細
```

## まとめ

このテストで確認すべき目次項目：
1. はじめに
2. テストの目的（はじめにの配下）
3. 正常な見出し1
4. 正常な見出し1-1（正常な見出し1の配下）
5. コードブロックのテスト
6. 正常な見出し2
7. 正常な見出し2-1（正常な見出し2の配下）
8. まとめ

コードブロック内の「これはコードブロック内の見出し」などは含まれないはずです。
